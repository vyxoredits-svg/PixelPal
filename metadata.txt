## Short Elevator Pitch
PixelPal is a premium, retro-modern browser companion game featuring programmatic 32x32 pixel canvas animations, custom 8-bit synthesizer tones, a fully realized item economy, quests, mini-games, and secure local-storage saves. Built as a Progressive Web App, players can install it directly on desktop and mobile and play completely offline.
