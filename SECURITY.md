# Security Policy

## Supported Versions

Only the latest release version receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| v2.2.x  | :white_check_mark: |
| < v2.2  | :x:                |

## Reporting a Vulnerability

If you discover a potential security vulnerability, please do NOT create a public issue. Instead, report it privately:

1. Open a private feedback channel or contact the repository owner.
2. Provide detailed reproduction steps.
3. We will respond within 48 hours to validate and patch the issue.
