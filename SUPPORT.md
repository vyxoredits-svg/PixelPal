# Support Guide

We want to ensure you have a cozy and smooth experience playing with PixelPal! Here is how to get help:

## 🔍 Self-Help Troubleshooting

- **CORS Errors / Blank Page locally**: PixelPal uses ES Modules (`type="module"`), which security policies block when opened via `file://`. Make sure to run it using a local server (`npx serve`, Live Server in VS Code, or python's `http.server`).
- **Save issues**: PixelPal automatically validates and repairs corrupt states. If you run into issues, try exporting your save via the Statistics panel, clearing site data, and importing the JSON file.

## 💬 Community Support

- **Bug Reports & Feedback**: Please open a GitHub Issue using the templates.
- **Feature ideas**: Post them in the repository Discussions tab.
