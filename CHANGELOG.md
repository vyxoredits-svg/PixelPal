# Changelog - PixelPal

All notable changes to PixelPal will be documented in this file.

## [2.1] - 2026-06-25
### Added
- **Volume slider control**: Interactive range input added in header to adjust retro chimes.
- **Export & Import Save Actions**: Backup progress to JSON files and upload to restore.
- **Save Validation & Repair**: Corrupt fields are automatically repaired with safe defaults during load.
- **Onboarding Tutorial Overlay**: Step-by-step introduction modal spotlighting panels.
- **Developer Dashboard**: Hidden analytics and control panel toggled by clicking the version badge 5 times.
- **Credits Page & Release preparation**: Readme and Changelog documentations.

### Optimized
- **30FPS Rendering Throttle**: Throttled render cycles to 30fps to cut CPU/GPU usage in half on low-end devices.
- **DOM Selector Caching**: Cached HTML elements on startup to prevent repetitive layout calculations during tick loops.
- **Animation smoothness**: Tuned tail wags, hops, and wings flapping into comfortable, smooth arcs.

## [2.0] - 2026-06-25
### Added
- **Pet Shop & Inventory System**: Currency purchase loop with food items, persistent toys, and placeable furniture.
- **Memory, Gauge, and Click Mini-Games**: Memory Match, Catch the Fish, Chase the Yarn Ball.
- **Daily & Weekly Quests**: Progress trackers awarding coins and XP.
- **Friendship Ranks**: Level up relationships with Pixel.
- **Collections**: Checklists tracking items purchased.
- **Prestige Rebirth**: Reset at lvl 30 for permanent multipliers.

## [1.2] - 2026-06-25
### Added
- **32x32 Detailed Pixel Art**: Ears, paws, tails, whiskers, and cheeks added.
- **Growth Stages**: Kitten, Older Kitten, Adult, and Celestial Evolved appearances.
- **Environmental Upgrades**: Starry nights/day windows, weather effects (Rain/Snow), and room wallpaper upgrades.

## [1.1] - 2026-06-25
### Added
- Programmatic HTML5 canvas rendering engine.
- Save data persistence using LocalStorage.
- 8-bit Sound effects synthesizer.
