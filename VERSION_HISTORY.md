# Version History - PixelPal

Below is a detailed list of all versions, dates, and features released for PixelPal.

| Version | Date | Description | Key Additions |
| :--- | :--- | :--- | :--- |
| **v2.2.0** | June 25, 2026 | **PWA Release** | Added manifest.json, sw.js offline service worker caching, and app icons. |
| **v2.1.0** | June 25, 2026 | **Modular Refactor** | Split 2,700-line monolithic script.js into 9 distinct ES Modules. |
| **v2.0.0** | June 20, 2026 | **Visual & Gameplay Overhaul** | Programmatic pixel art canvas renderer, shop, inventory, minigames, quests, prestige, save hardening. |
| **v1.1.0** | June 10, 2026 | **Cozy Style Polish** | Initial CSS enhancements, glassmorphism, responsive grid, developer dashboard overlay, and interactive tutorial. |
| **v1.0.0** | June 05, 2026 | **Initial Prototype** | Original virtual pet companion prototype with core stats (Hunger, Energy, Happiness) and local storage saving. |
